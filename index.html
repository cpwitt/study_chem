Subscripts and Writing Formulas

Subscripts and Writing Formulas

Subscripts are a crucial part in writing chemical formulas. Using them incorrectly could alter a formula's representation of a compound. Subscripts show how many atoms of an atomic element are in the compound. To know which number to use, you must know the charge of the atoms. When properly writing chemical formulas, the charges must cancel out, meaning that you might have 2 positive atoms that each have a +1 charge combined with 1 negative atom that has -2 charge.

Examples:

  • Li2O → Lithium has a +1 charge and Oxygen has a -2 charge. You need 2 Lithium atoms to balance the charges.
  • Ca3P2 → Calcium has a +2 charge and Phosphorus has a -3 charge. To balance out the charge, you need 3 Calcium atoms and 2 Phosphorous atoms.
  • H2O → This says that there are 2 Hydrogen atoms in this compound. Notice how there isnt an O1. That would be redundant.
  • Na3P → 2 Sodium atoms and 1 Phosphorous atom.
  • Ra3Bi2 → 3 Radium atoms and 2 Bismuth atoms.
  • Ca(ClO)2 → 1 Calcium atom and 2 Hypochlorite ions.
  • ↑
  • Notice how there are parentheses around ClO. This is because ClO is a Polyatomic Ion. To learn more about the parentheses, refer to the Metals with Polyatomic Ions section.

Ionic Compounds

If the first element in the chemical compound is a metal, then it is an Ionic Compound. Ionic Compunds are formed when a metal and a nonmetal bond.

Metals with Nonmetals

If the metal is in group 1, 2, or 13, or is either Zinc, Silver, or Cadmium, then naming it is fairly simple. First, write the name is the metal. Then, put the root of the nonmetal with "ide" as the suffix.

Examples:

  • NaCl → Sodium Chloride
  • Ca3P2 → Calcium Phosphide
  • Rb2Se → Rubidium Selenide
  • Al2O3 → Aluminum Oxide

Transition Metals with Nonmetals

If your metal is a Transition Metal (except for Zinc, Cadmium, and Silver), then the nomenclature rules are fairly similar. First, put the metal. The metal will be followed by a Roman Numeral in parentheses to show its charge. Then, put the root of the nonmetal with "ide" as the suffix.

How do I Know the Charge of the Transition Metal?

To find the charge of the Transition Metal, you first need to find the charge of the nonmetal.

For Example:

  • FeO; You know that Oxygen has a -2 charge, meaning that Iron must have a +2 charge. Therefore, FeO → Iron (II) Oxide

Roman Numeral Rules

Roman numerals only apply to transition metals, since transition metals do not have a set charge. For example, If there is a +1 charge, write (I) after the transition metal. However, the roman numeral rule does not apply to Zinc, Silver, or Cadmium. This is because these transition metals are special and have a set charge. Silver always has a +1 charge and Zinc and Cadmium both always have +2 charges.

Examples:

  • ZCl2 → Zinc Chloride (notice how I did't use a roman numeral because of Zinc)
  • TiBr → Titanium (I) Bromide
  • VTe → Vanadium (II) Telluride
  • RhP → Rhodium (III) Phosphide

Metals with Polyatomic Ions

Similar to the Ionic Compounds consiting of metals and nonmetals, if the ionic compound consists of a metal in groups 1, 2, or 13, or is zinc, cadmium, or silver and a polyatomic ion, then just name the metal and then write the polyatomic ion.

For Example:

  • Na2CO3 → Sodium Carbonate
  • Al2(ZnO2)3 → Aluminum Zincate

How Would I Write the Chemical Formula if I Was Given a Name?

If you were given a name of a metal + polyatomic ion, writing the formula is simple. You are told to write Potassium Phosphate. You are given the chemical formula for phosphate, PO33-. Since you need to balance out the compound, you need 3 Potassiums (potassium has +1 charge) to cancel out the -3 charge. You would then get K3PO3. (You do not need to write the charge of the polyatomic when writing the formula). In addition, if you ever need more than one polyatomic ion, make sure to put the polyatomic in parentheses and notate the charge with a subscript. For example: (PO3)2. You will also notice this rule with Aluminum Zincate from the above example.

Transition Metals with Polyatomic Ions

If the Ionic Compound consists of a Transition Metal (except Zinc, Cadmium, or Silver) and a polyatomic ion, follow the same rules that apply for metals where you do know the charge. The only difference is that you need to put the roman numeral in parentheses to state the charge of the metal. If confused, refer back to the Metals with Nonmetals section.

Examples:

  • FeSO4 → Iron (II) Sulfate (you are given that Sulfate has a -2 charge)
  • CuAlO3 → Copper (III) Aluminate

Covalent Compounds

Covalent Compounds are compounds that consist of a metal and nonmetal. To write the names of the formulas, name the first element and then the root of the second element with "ide" at the end. To specify how many atoms of an element you are using, you must use these prefixes before the elements:

Prefixes

1 mono- 6 hexa-
2 di- 7 hepta-
3 tri- 8 octa-
4 tetra- 9 nona-
5 penta- 10 deca-

Note that you can use these prefixes before the first or second element, or both.

For Example:

  • NH3 → Nitrogen Trihydride
  • CH4 → Carbon Tetrahydride
    • Notice how these compounds exclude the 'mono' prefix. This is because the 'mono' prefix is redundant, and most chemists exclude it when naming compounds. However, one of the few exceptions to this is CO (Carbon Monoxide) or any other compound containing Oxygen.

Example With Oxygen:

  • H2O → while the common name is water, the formal name is Dihydrogen Monoxide.

Acids

If the first element in your chemical formula is a Hydrogen, then the compound is an acid. An acid is a molecule or other entity that can donate a proton or accept an electron pair in reactions. All acids contain Hydrogen atoms.

Acids Without Polyatomic Ions

If the acid does not contain a polyatomic ion, start naming it by putting a "hydro", then put the root of the element with "ic" as the ending. The second word should then be "acid".

Examples:

  • HCl → Hydrochloric Acid
  • H2S → Hydrosulfuric Acid
  • HBr → Hydrobromic Acid

Acids With Polyatomic Ions

If the acid contains a polyatomic ion, start naming it by putting the root of the polyatomic first, then use either "ic" or "ous" as a suffix, followed by the word "acid". Do not prefix the formula with "hydro". Remember that you will need to refer to your "Common Polyatomic Ions" sheet.

How do I know which suffix to use?

If the polyatomic ion ends in "ate", use "ic" as the suffix.

  • H2CO3 → Carbonic Acid

If the polyatomic ion ends in "ite", use "ous" as the suffix.

  • HNO2 → Nitrous Acid
